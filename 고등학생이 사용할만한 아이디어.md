# 고등학생이 돈 내고 사용할 만한 프로그램 아이디어 5가지

## 1. 오답노트 관리 앱
- 틀린 문제 사진 찍어서 저장
- 과목별/단원별 자동 분류
- 복습 주기 알림 (1일, 3일, 7일 후)
- 시험 전 틀린 문제만 모아보기

**핵심 가치**: 손으로 쓰는 오답노트보다 빠르고, 언제 어디서든 복습 가능

---

## 2. 스터디 그룹 타이머
- 친구들과 실시간 공부 시간 공유
- 그룹 내 공부 시간 랭킹
- 서로 응원 메시지/찌르기 기능
- 주간/월간 공부 시간 통계

**핵심 가치**: 혼자 공부하면 늘어지는데, 친구들과 경쟁하면 집중력 UP

---

## 3. 내신 등급 계산기 + 예측
- 현재 점수 입력 → 예상 등급 계산
- 목표 등급 달성을 위한 필요 점수 알려줌
- 지난 시험 등급컷 데이터 저장
- 과목별 성적 추이 그래프

**핵심 가치**: "몇 점 더 받아야 등급 오르지?" 바로 확인 가능

---

## 4. 수능 플래너 (D-Day + 공부 기록)
- 수능까지 남은 시간 실시간 표시
- 과목별 공부 시간 기록 및 분석
- 취약 과목 자동 알림 ("국어 공부 3일째 안 했어요!")
- 목표 대비 달성률 시각화

**핵심 가치**: 단순 D-Day가 아닌, 실제 공부 패턴 분석 및 피드백

---

## 5. 집중력 키우기 게임형 타이머
- 공부 시간만큼 캐릭터/나무가 성장
- 중간에 폰 만지면 캐릭터 피해 입음
- 친구와 캐릭터 성장 경쟁
- 일정 시간 달성 시 업적/보상 시스템

**핵심 가치**: 게임 요소로 공부 동기 부여, 폰 중독 방지

---

## 수익화 전략 제안

| 방식 | 설명 |
|------|------|
| 프리미엄 기능 | 기본 무료, 고급 통계/분석은 유료 |
| 구독 모델 | 월 1,900~3,900원 수준 |
| 광고 제거 | 무료 버전은 광고, 유료는 광고 없음 |
| 꾸미기 아이템 | 캐릭터 스킨, 테마 판매 |

---

## 성공 포인트

1. **친구와 함께** - 소셜/경쟁 요소 필수
2. **게임성** - 성장, 보상, 업적 시스템
3. **실제 도움** - 성적 향상에 직접 연결
4. **예쁜 디자인** - 고등학생 취향 UI/UX

1. **Forest**: 게임성은 좋지만 **친구와 함께하는 기능이 약함**

# 2가지 핵심 방향 집중 기획

## 방향 A: 소셜 게임형 스터디 앱

### 컨셉
> "친구들과 함께 캐릭터를 키우며 공부하는 앱"

### 앱 이름 후보
- 공부몬 (공부 + 몬스터/포켓몬)
- 스터딩 (Study + 育成)
- 같이품 (같이 + 열품타 느낌)

### 핵심 기능

| 기능 | 설명 |
|------|------|
| **내 캐릭터** | 공부할수록 성장하는 나만의 캐릭터 |
| **그룹 생성** | 카카오톡 링크로 친구 초대 (최대 5명) |
| **실시간 현황** | 지금 누가 공부 중인지 한눈에 확인 |
| **협동 미션** | 그룹 전체 10시간 달성 → 캐릭터 진화 |
| **보스 레이드** | 주간 목표 달성 시 보스전 도전 |
| **찌르기** | 친구가 3시간 이상 안 하면 알림 보내기 |

## 방향 B: 오답 RPG 앱

### 컨셉
> "틀린 문제가 몬스터가 되고, 복습하면 처치하는 학습 RPG"

### 앱 이름 후보
- 오답헌터
- 몬스터노트
- 복습던전

### 핵심 기능

| 기능 | 설명 |
|------|------|
| **오답 등록** | 틀린 문제 사진 → 자동으로 몬스터 생성 |
| **몬스터 도감** | 과목별 몬스터 (수학=골렘, 영어=드래곤 등) |
| **복습 전투** | 문제 다시 풀기 → 맞추면 몬스터 처치 |
| **던전 시스템** | 시험 범위 = 던전, 오답 0개 = 던전 클리어 |
| **보상 시스템** | 처치 시 골드/경험치 → 장비 구매 |
| **친구 랭킹** | 주간 몬스터 처치 수 랭킹 |
