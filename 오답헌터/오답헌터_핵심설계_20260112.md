# 오답헌터 - 핵심 설계서

> 내 오답이 몬스터가 된다 - 로그라이크 학습 RPG

---

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 장르 | 로그라이크 + 학습 RPG |
| 플랫폼 | 모바일 (Android / iOS) |
| 타겟 | 한국 고등학생 |
| 핵심 | 개인 오답 사진 → 몬스터 → 복습 전투 |
| 분위기 | 다크 + 네온 (게임 느낌) |

---

## 2. 핵심 차별점

### 2-1. 경쟁 앱과의 차이

| 요소 | 기존 학습 앱 | 오답헌터 |
|------|-------------|---------|
| 오답 소스 | 앱 DB 문제 | **내가 찍은 실제 오답** |
| 게임 구조 | 단순 퀴즈 반복 | **로그라이크 런** |
| 성장 시스템 | 없거나 단순 | **영구 강화 + 아이템 수집** |
| 분위기 | 밝음/귀여움 | **다크 + 네온** |
| 타겟 | 전 연령/애매함 | **한국 고등학생 특화** |

### 2-2. 4대 차별점

```
1. 📸 개인 오답 사진 기반
   → 내가 틀린 문제가 몬스터
   → 감정적 연결 + 복습 동기

2. 🎲 로그라이크 + 학습 결합
   → 매 런마다 다른 버프
   → 죽어도 성장하는 구조

3. 🇰🇷 한국 고등학생 타겟
   → 수능/내신 중심
   → 국내 시장 틈새

4. 🌙 다크/네온 분위기
   → "게임"처럼 보이는 학습 앱
   → 시각적 차별화
```

---

## 3. 게임 루프

### 3-1. 기본 흐름

```
[오답 촬영] → 몬스터 등록
      ↓
[던전 입장] → 내 오답 몬스터 대기
      ↓
[스테이지 1~10] → 문제 풀이 = 전투
      ↓
[보스 스테이지] → 클리어 or 사망
      ↓
[결과] → 골드/경험치/아이템
      ↓
[영구 강화] → 더 강해져서 재도전
```

### 3-2. 런 시스템

```
1런 = 던전 1회 도전 (5~15분)

[런 중 획득]
├── 임시 버프 → 이번 런만
├── 골드 → 영구 보유
├── 경험치 → 영구 보유
└── 아이템 → 영구 보유 (보스 드랍)

[런 실패해도]
├── 골드/경험치 유지
└── 영구 강화로 다음 런 유리

→ "죽어도 성장한다" = 반복 동기
```

### 3-3. 세션 설계

| 지표 | 목표 |
|------|------|
| 1런 소요 | 5~15분 |
| 일일 플레이 | 3회 이상 |
| 스테이지 수 | 10개 |

## 4. 오답 → 몬스터 시스템

### 4-1. 등록 프로세스

```
1. 📷 문제 촬영 (카메라/갤러리)
2. 🏷️ 과목 태그 선택
3. 📁 단원 선택 (선택사항)
4. 👾 몬스터 자동 생성
5. ⚔️ 던전에 추가
```

### 4-2. 과목별 몬스터

| 과목 | 몬스터 타입 | 컨셉 |
|------|------------|------|
| 수학 | 골렘/슬라임 | 숫자와 수식 |
| 영어 | 드래곤/유령 | 알파벳 문양 |
| 국어 | 책/두루마리 | 한자/고전 |
| 과학 | 실험체/로봇 | 비커/원소 |

### 4-3. 복습 기한 = 난이도

| 경과 일수 | 상태 | HP |
|-----------|------|-----|
| 0~3일 | 일반 | 100 |
| 4~7일 | 강화 | 150 |
| 8~14일 | 분노 | 200 |
| 15일+ | 보스급 | 300 |

→ 복습 미루면 몬스터 강해짐 = 복습 동기

## 5. 전투 시스템

### 5-1. 전투 화면 구성

```
┌─────────────────────────────┐
│ HP ████████░░  LV.12        │
│ 스테이지 3/10    💰 1,240G   │
├─────────────────────────────┤
│         👾                  │
│    [ 방정식 슬라임 ]         │
│      HP ██████░░░░          │
├─────────────────────────────┤
│ ┌─────────────────────────┐ │
│ │ 📸 내가 찍은 오답 이미지  │ │
│ │   x² - 5x + 6 = 0       │ │
│ └─────────────────────────┘ │
│                             │
│  [2] [3] [2,3] [-2]  ⏱️ 23초 │
├─────────────────────────────┤
│ [💡힌트 50G] [⏭️스킵] [⏸️]  │
└─────────────────────────────┘
```

### 5-2. 전투 규칙

| 행동 | 결과 |
|------|------|
| ✅ 정답 | 몬스터 데미지 + 골드/경험치 |
| ❌ 오답 | 플레이어 HP 감소 + 콤보 리셋 |
| ⏱️ 시간 초과 | 플레이어 HP 감소 |
| 몬스터 HP 0 | 스테이지 클리어 → 버프 선택 |
| 플레이어 HP 0 | 런 종료 → 영구 강화 화면 |

## 6. 보상 시스템

### 6-1. 난이도별 기본 보상

| 난이도 | 골드 | 경험치 |
|--------|------|--------|
| ⭐ 기초 | 10G | 5 exp |
| ⭐⭐ 보통 | 25G | 15 exp |
| ⭐⭐⭐ 어려움 | 50G | 35 exp |
| ⭐⭐⭐⭐ 심화 | 100G | 70 exp |
| ⭐⭐⭐⭐⭐ 킬러 | 200G | 150 exp |

### 6-2. 타임 보너스

| 풀이 시간 | 보너스 |
|-----------|--------|
| 30% 이내 | +100% |
| 50% 이내 | +50% |
| 80% 이내 | +20% |

### 6-3. 콤보 시스템

```
연속 정답 배율:
1문제: x1.0
3문제: x1.2
5문제: x1.5 🔥
10문제: x2.0 🔥🔥
20문제: x3.0 🔥🔥🔥

→ 틀리면 콤보 리셋
```

## 7. 던전 시스템

### 7-1. 던전 구조

```
[수학 던전]
├── 스테이지 1~3: 기초
├── 스테이지 4~6: 보통
├── 스테이지 7~9: 어려움
└── 스테이지 10: 보스

클리어 시 → 다음 난이도 해금
```

### 7-2. 던전 종류

| 던전 | 해금 조건 | 문제 유형 |
|------|-----------|-----------|
| 수학 | 기본 | 계산, 방정식, 함수 |
| 영어 | LV.5 | 단어, 문법, 독해 |
| 국어 | LV.10 | 문학, 비문학, 문법 |
| 과학 | LV.15 | 물리, 화학, 생물 |
| 혼합 | LV.20 | 전 과목 랜덤 |

### 7-3. 적응형 난이도

```
최근 10문제 정답률 기준:

90% 이상 → 난이도 UP
70~90%  → 유지
50~70%  → 힌트 제공
50% 미만 → 난이도 DOWN

→ 항상 적절한 도전 유지
```
