# 오답헌터 - 시스템 설계서

> 로그라이크, 아이템, 캐릭터, UI/UX 상세 설계

---

## 1. 로그라이크 시스템

### 1-1. 런 중 버프 선택

```
[스테이지 클리어 시]

┌─────────┐ ┌─────────┐ ┌─────────┐
│ 🗡️ 공격  │ │ 🛡️ 방어  │ │ ⏱️ 시간  │
│ +20%    │ │ +1 목숨  │ │ +10초   │
└─────────┘ └─────────┘ └─────────┘

→ 3개 중 1개 선택
→ 이번 런에만 적용
→ 매 런마다 다른 빌드
```

### 1-2. 버프 종류

| 카테고리 | 버프 | 효과 |
|----------|------|------|
| 공격 | 데미지 +20% | 정답 시 데미지 증가 |
| 공격 | 크리티컬 +15% | 확률적 2배 데미지 |
| 공격 | 관통 | 오버킬 데미지 다음 적에게 |
| 방어 | HP +30 | 최대 HP 증가 |
| 방어 | 부활 1회 | HP 0 → 50% 부활 |
| 방어 | 피해 감소 -20% | 오답 시 데미지 감소 |
| 시간 | 시간 +10초 | 문제당 제한시간 증가 |
| 시간 | 시간 정지 | 3회 일시정지 가능 |
| 특수 | 골드 +30% | 획득 골드 증가 |
| 특수 | 경험치 +30% | 획득 경험치 증가 |
| 특수 | 힌트 무료 | 이번 런 힌트 무료 |

### 1-3. 랜덤 이벤트

```
[?] 스테이지 5 이벤트

"수상한 상인이 나타났다"

[50G 지불] → 랜덤 버프 획득
[무시하기] → 패스

---

[?] 스테이지 8 이벤트

"희미한 빛이 보인다"

[조사하기] → 50% 버프 / 50% 함정
[무시하기] → 패스
```

| 이벤트 | 선택지 | 결과 |
|--------|--------|------|
| 상인 | 골드 지불 | 랜덤 버프 |
| 빛 | 조사 | 50% 버프/함정 |
| 보물상자 | 열기 | 골드 or 몬스터 |
| 휴식처 | 휴식 | HP 30% 회복 |

### 1-4. 영구 강화 (메타 진행)

```
[런 종료 후]

💀 런 실패...
획득: 120G, 85exp

[영구 강화 상점]
├── 기본 HP +10 .......... 200G
├── 시작 시간 +5초 ........ 150G
├── 골드 획득량 +10% ...... 300G
├── 경험치 획득량 +10% .... 300G
├── 시작 버프 슬롯 +1 ..... 500G
└── 새 캐릭터 해금 ........ 500G
```

## 2. 아이템 시스템

### 2-1. 획득 방법

| 방법 | 확률 | 비고 |
|------|------|------|
| 보스 처치 | 100% | 1개 드랍 |
| 엘리트 처치 | 30% | 스테이지 5, 8 |
| 상자 이벤트 | 20% | 랜덤 이벤트 |

### 2-2. 희귀도

| 등급 | 색상 | 드랍률 | 특징 |
|------|------|--------|------|
| 일반 | ⚪ | 60% | 기본 효과 |
| 레어 | 🔵 | 25% | 강화된 효과 |
| 에픽 | 🟣 | 12% | 특수 효과 |
| 전설 | 🟡 | 3% | 고유 효과 |

### 2-3. 장비 슬롯

```
[캐릭터 장비]

┌─────────────────────┐
│      👤 캐릭터       │
├─────────────────────┤
│ 🗡️ 무기: X의 검     │
│ 🛡️ 방어구: 공식의 갑옷│
│ 💍 악세서리: 계산기 반지│
└─────────────────────┘
```

| 슬롯 | 주요 효과 |
|------|-----------|
| 무기 | 공격력, 골드 획득, 데미지 |
| 방어구 | HP, 방어력, 피해 감소 |
| 악세서리 | 특수 효과, 확률 |

### 2-4. 주요 아이템 목록

**수학 던전 드랍**
| 아이템 | 등급 | 효과 |
|--------|------|------|
| X의 검 | 🔵 | 수학 시간 +5초 |
| 공식의 갑옷 | 🔵 | 수학 피해 -15% |
| 계산기 반지 | 🟣 | 계산 문제 힌트 무료 |
| 적분왕의 왕관 | 🟡 | 수학 골드 2배 |

**영어 던전 드랍**
| 아이템 | 등급 | 효과 |
|--------|------|------|
| Grammar Shield | 🔵 | 오답 데미지 -20% |
| Vocab Ring | 🟣 | 보기 1개 제거 |
| Reading Eye | 🟡 | 지문 핵심 하이라이트 |

**전설 아이템**
| 아이템 | 효과 |
|--------|------|
| ⚔️ 만점자의 검 | 정답 시 데미지 +50% |
| 👑 1등급의 왕관 | 10콤보마다 HP 회복 |
| 💎 완벽의 보석 | 런 시작 시 버프 3개 |

### 2-5. 세트 효과

```
[수학 마스터 세트]

🗡️ X의 검 ✅
🛡️ 공식의 갑옷 ✅
💍 계산기 반지 ❌

(2세트) 수학 시간 +3초
(3세트) 수학 난이도 -1
```

| 세트 | 효과 |
|------|------|
| 수학 마스터 | 수학 던전 특화 |
| 영어 학자 | 영어 던전 특화 |
| 콤보 킹 | 콤보 유지 보너스 |
| 타임 어택 | 시간 보너스 강화 |

### 2-6. 강화 시스템

```
중복 아이템 획득 시 → 자동 강화

X의 검 → X의 검 +1 → X의 검 +2 → X의 검 +3 (MAX)
(+5초)    (+6초)       (+8초)       (+10초)
```

### 2-7. 컬렉션 도감

```
┌─────────────────────────────┐
│  📚 아이템 도감              │
│  수집률: 34/120 (28%)       │
│  ████░░░░░░░░░░░            │
├─────────────────────────────┤
│  [무기] 12/30               │
│  ⚪⚪⚪🔵🔵🟣❓❓❓🟡      │
│                             │
│  [방어구] 8/25              │
│  ⚪⚪⚪⚪🔵❓❓❓❓❓       │
│                             │
│  ❓ = 미획득 (실루엣)       │
└─────────────────────────────┘

→ 수집 보상: 도감 % 달성 시 보너스
→ 10%: 골드 +5%
→ 50%: 특별 칭호
→ 100%: 히든 캐릭터 해금
```

## 3. 캐릭터 시스템

### 3-1. 캐릭터 목록

| 캐릭터 | 해금 | 특징 | 기본 스탯 |
|--------|------|------|-----------|
| 기본 용사 | 시작 | 밸런스 | HP 100, 시간 30초 |
| 속도형 | 300G | 시간+, HP- | HP 70, 시간 45초 |
| 탱커형 | 300G | HP+, 시간- | HP 150, 시간 20초 |
| 도박형 | 500G | 랜덤 | 매 런 스탯 랜덤 |
| ??? | 히든 | 특수 | 특수 조건 해금 |

### 3-2. 레벨 시스템

```
레벨업 = 기본 스탯 상승

LV.1 → LV.10: HP +5/레벨, 시간 +1초/레벨
LV.10 → LV.20: HP +3/레벨, 시간 +0.5초/레벨
LV.20+: HP +2/레벨

레벨업 필요 경험치:
LV × 100 + (LV-1) × 50

예시:
LV.1 → 2: 100 exp
LV.5 → 6: 700 exp
LV.10 → 11: 1,450 exp
```

### 3-3. 히든 캐릭터 해금 조건

| 캐릭터 | 조건 |
|--------|------|
| 올클리어 | 모든 던전 클리어 |
| 수집가 | 도감 100% |
| 콤보왕 | 50콤보 달성 |
| 무피해 | 피해 0으로 보스 클리어 |

## 4. UI/UX 가이드

### 4-1. 컬러 팔레트

```
[메인 컬러]
배경: #0a0a0f (거의 검정)
카드: #1a1a24 (어두운 회색)

[강조 컬러]
포인트: #6366f1 (보라/네온)
서브: #818cf8 (밝은 보라)

[기능 컬러]
성공: #22c55e (초록)
위험: #ef4444 (빨강)
경고: #fbbf24 (노랑)

[희귀도]
일반: #ffffff
레어: #3b82f6
에픽: #a855f7
전설: #fbbf24
```

### 4-2. 분위기 가이드

```
✅ DO
- 다크 모드 기본
- 네온 포인트 컬러
- 묵직한 효과음
- 세련된 폰트
- 미니멀한 UI
- 부드러운 애니메이션

❌ DON'T
- 밝은 색상
- 귀여운/아기자기한 느낌
- 복잡한 UI
- 과한 이펙트
```

## 6. 수익화

### 6-1. 모델

| 방식 | 설명 |
|------|------|
| 기본 무료 | 핵심 기능 무료 |
| 광고 | 런 사이 광고 (스킵 가능) |
| 광고 제거 | 3,900원 1회 구매 |
| 스킨 판매 | 캐릭터 스킨 500~2,000원 |
| 시작 팩 | 골드 + 아이템 (5,900원) |

### 6-2. 골드 사용처

| 항목 | 가격 |
|------|------|
| 힌트 | 50G |
| 부활권 | 100G |
| 캐릭터 해금 | 300~500G |
| 영구 강화 | 150~500G |
