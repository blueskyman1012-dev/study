# 중학교 3학년이 만들기 쉬운 프로그램 아이디어 10가지

## 1. 간단한 계산기
- 덧셈, 뺄셈, 곱셈, 나눗셈 기능
- Python이나 Scratch로 쉽게 구현 가능

## 2. 숫자 맞추기 게임
- 컴퓨터가 랜덤 숫자를 정하면 사용자가 맞추는 게임
- "더 높아요/낮아요" 힌트 제공

## 3. 가위바위보 게임
- 컴퓨터와 대결하는 가위바위보
- 승/패/무승부 기록 기능 추가

## 4. D-Day 계산기
- 특정 날짜까지 남은 일수 계산
- 시험일, 생일, 방학까지 남은 날 확인

## 5. 단어 퀴즈 프로그램
- 영어 단어 뜻 맞추기
- 점수 기록 기능 포함

## 6. BMI 계산기
- 키와 몸무게 입력 → BMI 수치 및 판정 결과 출력
- 건강 관련 메시지 표시

## 7. 타이머/스톱워치
- 공부 시간 측정용 타이머
- 알람 기능 추가 가능

## 8. 간단한 To-Do 리스트
- 할 일 추가/삭제/완료 체크
- 일정 관리 연습

## 9. 운세/랜덤 뽑기 프로그램
- 오늘의 운세, 점심 메뉴 추천
- 랜덤 선택 기능 활용

## 10. 구구단 게임
- 랜덤 구구단 문제 출제
- 정답률 및 걸린 시간 측정

---

## 추천 프로그래밍 언어
- **Python** - 문법이 간단하고 배우기 쉬움
- **Scratch** - 블록 코딩으로 시각적으로 쉽게 제작
- **App Inventor** - 스마트폰 앱 만들기 가능
