# 오답헌터 - 빠른 시작 가이드

## 1. 서버 실행

```bash
cd C:\hjkim\app
npm run dev
```

또는 `start.bat` 더블클릭

---

## 2. 접속

| 기기 | 주소 |
|------|------|
| PC | https://localhost:9333 |
| 모바일 | https://192.168.0.192:9333 |

> 모바일은 PC와 같은 WiFi에 연결 필요

---

## 3. 첫 오답 등록

1. **📸 오답 등록** 터치
2. 과목 선택 (수학/영어/과학/국어)
3. 카메라로 틀린 문제 촬영
4. OCR 분석 대기 (첫 실행 시 10-20초)
5. 문제 텍스트 확인/수정
6. 정답 입력
7. 몬스터 등록 완료!

---

## 4. 던전 입장

1. **⚔️ 던전 입장** 터치
2. 몬스터가 출제하는 문제 풀기
3. 4지선다 중 정답 선택
4. 30초 내 정답 선택
5. 100층 클리어!

---

## 5. AI 연동 (선택)

### API 키 발급
1. https://aistudio.google.com 접속
2. Google 로그인
3. **Get API Key** → **Create API Key**
4. 키 복사

### 게임에 적용
1. **⚙️ 설정** 터치
2. **🔑 API 키 입력** 터치
3. 키 붙여넣기
4. **테스트 문제 생성**으로 확인

### AI 기능
- 오답 시 유사 문제 자동 생성
- 문제 유형 분석
- 풀이 힌트 제공

---

## 6. 게임 팁

### 전투
| 행동 | 결과 |
|------|------|
| 정답 | 몬스터 데미지 + 골드 획득 |
| 오답 | 내 HP 감소 + 콤보 초기화 |
| 시간초과 | 내 HP -15 |
| 콤보 유지 | 데미지/골드 증가 |

### 보스
| 층 | 보스 |
|----|------|
| 10, 20, 30... | 👹 일반 보스 |
| 50 | 👿 중간 보스 |
| 100 | 🐉 최종 보스 |

### 아이템
- 일반 몬스터: 5% 드랍
- 보스: 35~100% 드랍
- 등급: 일반 < 레어 < 에픽 < 전설

## 문제 해결

### 모바일 접속 안됨
1. PC와 같은 WiFi 연결 확인
2. 방화벽 9333 포트 허용 확인
3. `http` 대신 `https` 사용

### 카메라 안 열림
1. HTTPS 접속 확인
2. 브라우저 카메라 권한 허용
3. iOS: "고급" → "이 웹사이트 방문" 선택

### OCR 느림
- 첫 실행 시 모델 다운로드 (10-20초)
- 이후 분석은 5초 내외

### 효과음 안 들림
1. 설정에서 효과음 ON 확인
2. 기기 볼륨 확인
3. 처음 화면 터치 후 소리 활성화
