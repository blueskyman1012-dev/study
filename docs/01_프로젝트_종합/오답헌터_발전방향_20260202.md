# 오답헌터 발전 방향 로드맵

## 현재 상태 요약
- 로그라이크 학습 RPG (수학/과학 2과목)
- 100층 던전, 3단계 보스, 99레벨 시스템
- AI 문제 분석/생성 (SmilePrint + Gemini)
- 4개 언어 지원 (한/영/일/중)
- Cloudflare D1 서버 동기화

---

## 1단계: 핵심 학습 기능 강화

### 1-1. 오답노트 & 복습 시스템
- **간격 반복 학습 (Spaced Repetition)**
  - 틀린 문제를 1일 → 3일 → 7일 → 30일 주기로 자동 재출제
  - 정답률에 따라 주기 조절 (잘 맞추면 간격 늘리기)
- **오답노트 화면**
  - 틀린 문제 모아보기, 과목/난이도/날짜별 필터
  - 풀이 과정과 해설 한눈에 보기
  - "다시 풀기" 버튼으로 즉시 재도전

### 1-2. 과목 확장
- **영어**: 문법, 어휘, 독해
- **국어**: 문학, 비문학, 문법
- **사회/역사**: 지리, 한국사, 세계사
- 과목별 전용 던전 테마 및 보스 디자인

### 1-3. 문제 유형 다양화
- 현재: 4지선다 위주
- 추가: 단답형, O/X, 빈칸 채우기, 순서 배열, 드래그&드롭
- 서술형 문제: AI가 답안 채점 (키워드 기반 + Gemini 평가)

---

## 2단계: 게임성 강화

### 2-1. 새로운 던전 시스템
- **일일 던전**: 매일 다른 테마, 한정 보상
- **무한 던전**: 끝없이 이어지는 도전 모드, 리더보드
- **이벤트 던전**: 시험 기간 특별 던전 (중간고사/기말고사)
- **타임 어택**: 제한 시간 내 최대 클리어

### 2-2. 장비 & 스킬 시스템
- **장비**: 무기(공격력↑), 방어구(HP↑), 악세서리(특수효과)
  - 장비 강화 (+1 ~ +10), 실패 시 파괴 확률
  - 세트 효과 (같은 시리즈 2~3개 장착 시 보너스)
- **스킬**: 전투 중 사용 가능한 능력
  - 힌트 스킬 (선택지 2개 제거)
  - 타임 프리즈 (10초간 시간 정지)
  - 더블 데미지 (다음 정답 2배 데미지)
  - 흡혈 (정답 시 HP 회복)

### 2-3. 펫 시스템
- 학습 도우미 캐릭터
- 정답률에 따라 성장 (알 → 유아 → 성체)
- 패시브 효과 (골드 보너스, 경험치 보너스, 힌트 자동 사용 등)

### 2-4. 업적 & 칭호
- 누적 업적: "100문제 정답", "연속 10콤보", "보스 무피격 클리어"
- 칭호 시스템: 프로필에 표시, 특수 효과 부여
- 일일/주간 미션: "오늘 수학 20문제 풀기"

---

## 3단계: 소셜 & 경쟁 기능

### 3-1. 리더보드
- 주간/월간/전체 랭킹
- 과목별, 난이도별, 레벨별 분류
- 학교/반별 그룹 랭킹

### 3-2. 친구 시스템
- 친구 추가 & 프로필 조회
- 친구 문제 공유 (내 오답을 친구에게 보내기)
- 친구 랭킹 비교

### 3-3. 대전 모드
- **1:1 실시간 대전**: 같은 문제를 동시에 풀어 속도/정확도 경쟁
- **길드전**: 길드원들의 정답률 합산으로 경쟁
- **토너먼트**: 주말 특별 이벤트

### 3-4. 문제 공유 마켓
- 사용자가 만든 문제 업로드
- 다운로드 수/좋아요 기반 추천
- 과목/학년별 카테고리

---

## 4단계: AI 고도화

### 4-1. 학습 분석 AI
- **취약 분야 자동 진단**: 오답 패턴 분석 → "너는 방정식의 이항에서 실수가 많아"
- **맞춤 학습 경로**: 레벨에 맞는 문제 난이도 자동 조절
- **학습 리포트**: 주간/월간 학습 통계, 성장 그래프

### 4-2. AI 튜터
- 오답 시 AI가 단계별 풀이 힌트 제공
- "왜 틀렸는지" 자동 분석 & 유사 개념 설명
- 음성 설명 (TTS 연동)

### 4-3. 이미지 인식 고도화
- 손글씨 수식 인식 정확도 향상
- 그래프/도형 문제 인식
- PDF/교과서 페이지 일괄 등록

### 4-4. AI 문제 생성 고도화
- 교과서 단원별 문제 자동 생성
- 난이도 세분화 (현 3단계 → 10단계)
- 함정 선택지 생성 (학생이 자주 틀리는 유형 기반)

---

## 5단계: 플랫폼 확장

### 5-1. PWA 완성
- 오프라인 모드: 인터넷 없이도 플레이 가능
- 푸시 알림: "복습할 시간이에요!"
- 홈 화면 설치 유도 배너

### 5-2. 네이티브 앱
- React Native 또는 Flutter로 전환
- 카메라 네이티브 연동 (촬영 품질 향상)
- 앱스토어/플레이스토어 배포

### 5-3. 학교/학원 관리자 대시보드
- 선생님 계정: 반 학생들의 학습 현황 모니터링
- 과제 출제: 특정 문제 세트를 학생들에게 배정
- 학습 리포트 다운로드 (PDF)

### 5-4. 학부모 모드
- 자녀의 학습 현황 실시간 확인
- 학습 시간/정답률 알림
- 주간 리포트 자동 발송 (이메일)

---

## 6단계: 수익화 & 지속 가능성

### 6-1. 프리미엄 플랜
- 무료: 일 20문제, 기본 던전, 광고 포함
- 프리미엄: 무제한 문제, 모든 던전, 광고 제거, AI 튜터
- 학교/학원 플랜: 관리자 대시보드 + 대량 라이선스

### 6-2. 인앱 결제
- 코스메틱: 캐릭터 스킨, 펫 외형, 이펙트
- 편의: 추가 인벤토리, 경험치 부스트
- (P2W 방지: 전투 능력에 직접 영향 X)

### 6-3. 교육 기관 파트너십
- 교과서 출판사 연동 (단원별 문제 DB)
- 학교 시스템(NEIS 등) 연동
- 교육청/EBS 콘텐츠 제휴
