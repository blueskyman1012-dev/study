# 문제 미풀이 포기 시 통계 미기록 수정

## 날짜
2026-02-23

## 문제점
문제를 하나도 풀지 않고 포기 버튼을 눌러도 통계에 "실패"로 기록되어 승률 등 통계가 왜곡됨.

## 수정 내용

### `app/src/game/Game.js` - `endRun()` 함수
- 포기(`isWin === false`)이고 `totalAnswers === 0`(한 문제도 풀지 않음)일 때:
  - DB 저장 (`db.add('runs', ...)`) 건너뜀
  - 통계 캐시 무효화 건너뜀
  - 서버 전송 (`apiService.postRun()`) 건너뜀
  - 업적 처리 (`achievementManager.onRunEnd()`) 건너뜀
- HP 초기화, 몬스터 부활, 결과 화면 이동은 정상 수행

## 검증
- `npm run build` 성공
- `npx vitest run` 전체 통과 (8 파일, 209 테스트)
