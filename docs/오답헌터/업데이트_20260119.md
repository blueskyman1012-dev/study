# 오답헌터 업데이트 (2026-01-19)

## 1. 적응형 난이도 시스템

### 정답률 추적
- `currentRun`에서 정답/총문제 수 추적
- 5문제 이상 풀면 적응형 난이도 적용

### 난이도 조절 규칙
| 정답률 | 효과 |
|--------|------|
| 80% 이상 | 어려운 문제 70% 확률 |
| 50~80% | 랜덤 출제 |
| 50% 미만 | 쉬운 문제 70% 확률 |

### UI 표시
- 전투 화면: 정답률 % 표시 (80%↑ 빨강, 50%↓ 초록)
- 결과 화면: 최종 정답률 (정답/총문제)

---

## 2. 시각 효과 시스템

### 이펙트 종류
| 효과 | 설명 |
|------|------|
| 파티클 | 몬스터 처치, 콤보 5+ 시 폭발 |
| 플로팅 텍스트 | 데미지, 골드, 콤보 숫자 |
| 화면 흔들림 | 오답, 보스 등장 시 |
| 화면 플래시 | 정답(초록), 오답(빨강), 처치(흰색) |

### 정답/오답 연출
- **정답**: 초록 플래시 + 데미지 숫자
- **오답**: 빨강 플래시 + 화면 흔들림 + 피해 숫자
- **콤보 3+**: 콤보 텍스트 표시
- **콤보 5+**: 파티클 폭발 추가

### 보스 연출
- 등장: 화면 흔들림 + 플래시 + 아이콘 확대
- 테두리: 펄스 애니메이션
- 처치: 등급별 파티클 (최종보스 50개)

### 몬스터 애니메이션
- 호흡 효과 (배경 원 확대/축소)
- 보스 입장 줌인

## 3. HP바/타이머 개선

### HP바
- 3단계 그라데이션 (밝음→기본→어두움)
- 상단 하이라이트 효과

### 타이머 바
- 시간별 색상 변화: 초록→노랑→빨강
- 25% 이하 깜빡임 경고
