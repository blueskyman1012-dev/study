## 구현 일자
2026-02-02

## 개요
업적(25개, 영구 해제)과 일일 미션(매일 3개, 랜덤) 시스템을 구현하였습니다.

## 새로 생성한 파일
1. **app/src/game/AchievementManager.js** - 업적 체크/해제, 일일 미션 관리
2. **app/src/game/screens/AchievementScreen.js** - 업적 화면 (스크롤 지원)

## 수정한 파일
1. **app/src/utils/constants.js** - SCREENS.ACHIEVEMENT, ACHIEVEMENTS(25개), DAILY_MISSIONS(10개 풀) 추가
2. **app/src/i18n/i18n.js** - 4개 언어 약 280개 번역 키 추가
3. **app/src/game/Game.js** - AchievementManager 초기화, 화면 라우팅, endRun/등록 훅
4. **app/src/game/BattleManager.js** - onCorrectAnswer, onGoldEarned, onMonsterDefeated 훅
5. **app/src/game/PlayerManager.js** - 새 필드(achievements, dailyMissions 등), onLevelUp 훅
6. **app/src/game/screens/MainScreen.js** - 업적 버튼 추가 (4버튼 레이아웃)
7. **app/src/game/EffectSystem.js** - 업적 해제 배너 애니메이션
