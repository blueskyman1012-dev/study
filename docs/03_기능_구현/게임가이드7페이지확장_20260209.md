# 게임 가이드 7페이지 확장 구현

## 작업 일자
2026-02-09

## 개요
기존 4페이지 가이드를 7페이지로 확장하여 게임의 핵심 메커니즘을 상세히 설명하도록 개선.

## 변경 파일

### 1. `app/src/game/GuideManager.js`
- `totalPages`: 4 → 7
- `icons` 배열: `['👋', '📸', '⚔️', '💥', '🎒', '🏪', '🏆']`
- 줄 간격: 24px → 22px (더 많은 텍스트 수용)
- 페이지 번호 표시 추가: `"3/7"` 형태 (제목 아래 반투명 텍스트)

### 2. `app/src/i18n/i18n.js`
4개 언어(ko, en, ja, zh) 가이드 번역 키 업데이트

## 페이지 구성

| 페이지 | 아이콘 | 제목 | 내용 요약 |
|---|---|---|---|
| 0 | 👋 | 환영 | 게임 핵심 루프 소개 |
| 1 | 📸 | 오답 등록 | 카메라, AI 분석, 과목 선택 |
| 2 | ⚔️ | 던전 구조 | 100층, 보스 배치, 난이도 |
| 3 | 💥 | 전투 메커니즘 | 정답=공격, 오답=피해, 콤보 |
| 4 | 🎒 | 아이템 (신규) | 전투 아이템 + 드랍 시스템 |
| 5 | 🏪 | 상점 (신규) | 강화/소비/꾸미기 3탭 |
| 6 | 🏆 | 성장 (신규) | 레벨100, 업적, 일일/주간 미션 |
