# 문제 크게 보기 지원

## 수정일: 2026-02-09
## 수정 파일
- `app/src/game/screens/BattleScreen.js`
- `app/src/game/BattleManager.js`
- `app/src/game/DialogManager.js`

---

## 변경 내용

### 기존 문제
- 문제가 2줄 이상(44자 초과)일 때만 "전체보기" 버튼이 표시됨
- 짧은 문제도 13px 폰트로 작게 보여 확인이 불편할 수 있음

### 변경사항

#### 1. 항상 크게 보기 가능 (BattleScreen.js)
- 문제 길이와 무관하게 항상 🔍 아이콘 표시 (우측 상단)
- 문제 카드 전체 영역(360×128px)을 클릭 가능하게 설정
- 문제 카드 아무 곳이나 터치하면 크게 보기 모달 표시

#### 2. 전용 문제 모달 (DialogManager.showQuestionModal)
- 기존 범용 showModal(15px) 대신 전용 모달 신규 추가
- 문제 텍스트: **20px bold** (기존 대비 약 1.5배 크기)
- 선택지도 함께 표시: 각 선택지를 카드 형태로 16px 표시
- 주제(topic) 정보가 있으면 상단에 표시
- word-break: keep-all로 단어 단위 줄바꿈

#### 3. BattleManager.showFullQuestion 수정
- 기존: `game.showModal(message)` 호출
- 변경: `game.dialogManager.showQuestionModal(question, topic, choices)` 호출
- 선택지 데이터도 전달하여 모달에서 함께 표시
