# 상점 영구강화 진척도 UI 개선

> 작성일: 2026-02-09

---

## 변경 내용

### 기존 문제점
- 프로그레스 바가 8px 높이의 얇은 막대로 진척도 파악이 어려움
- 레벨 마커가 1.5px 크기의 점으로 거의 보이지 않음
- 모든 강화가 동일한 인디고색으로 구분이 안 됨

### 개선 사항 — 원형 프로그레스 링 방식으로 전면 교체

#### 1. 원형 프로그레스 링
- 기존 직선 프로그레스 바 완전 제거
- 아이콘 주위에 **원형 링(radius 26, lineWidth 5)** 으로 진행도 표시
- 12시 방향에서 시작, 시계 방향으로 채워짐
- 링 끝단 `lineCap: round` 로 부드러운 처리
- 링 뒤에 **글로우(shadowBlur) 효과** 추가

#### 2. 강화별 고유 색상 테마
| 강화 | 색상 | 코드 |
|------|------|------|
| HP | 초록 | `#22c55e` |
| 공격력 | 빨강 | `#ef4444` |
| 시간 | 파랑 | `#3b82f6` |
| 골드보너스 | 노랑 | `#fbbf24` |
- 링, 글로우, 레벨 뱃지, 악센트 바 모두 해당 색상으로 통일
- MAX 달성 시 전체 초록색 전환

#### 3. 링 중앙에 퍼센트/MAX 표시
- 아이콘 아래에 `33%` 또는 `MAX` 텍스트 표시
- 해당 강화의 고유 색상으로 표시

#### 4. 레벨 뱃지 개선
- `Lv.3` → `Lv.3/10` 으로 분모 포함
- 뱃지 너비를 텍스트에 맞게 동적 계산

#### 5. 카드 높이 최적화
- 카드 높이 116px → 100px 로 줄여 더 많은 항목이 한 화면에 표시
