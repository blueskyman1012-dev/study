## 변경 요약

### Tab 0 (종합) 개선
1. **3열 카드 → 2열 카드로 변경**
   - 기존: 승/패(114px) + 연승(114px) + 시간(114px) → 비좁아서 가독성 떨어짐
   - 변경: 승/패+연승(175px) 좌측 + 시간+플레이(175px) 우측으로 통합
   - 연승 정보는 좌측 카드 하단에 `🔥 현재/최대` 형태로 간결하게 표시
   - 총 플레이 횟수는 우측 시간 카드 하단으로 이동

2. **골드+아이템 카드 간결화**
   - 기존: 2행 3열 아이템 배치 (110px 높이) → 빽빽함
   - 변경: 골드를 상단에 크게 (22px bold), 아이템은 아이콘+숫자만 1행 배치 (80px 높이)
   - label 텍스트 제거하고 아이콘만으로 식별

### Tab 1 (과목) 개선
1. **섹션 A에 정답/오답 수 통합**
   - 기존 문제 수 표시 위치에 `✅15 ❌3` 형태로 정답/오답 수를 직접 표시
   - `correctBySubject`/`wrongBySubject` 데이터를 섹션 A에서 바로 활용

2. **섹션 B (과목별 오답 분석) 삭제**
   - 섹션 A와 중복되는 데이터를 다른 형태로 보여줄 뿐이었음
   - 정답/오답 정보가 A에 통합되어 불필요해짐

3. **섹션 C(유형별 상세) → 섹션 B로 승격**
   - 중복 섹션 제거로 핵심 기능인 유형별 상세가 더 위로 올라와 바로 보임
   - 미사용 변수 `subjectInfoC` 제거

### Tab 2 (분석) 개선
1. **최근 런 목록에 컬럼 헤더 추가**
   - 날짜 / 정답률 / 콤보 / 골드 / 시간 헤더 표시
   - 기존: 숫자만 나열 → 변경: 각 열이 무엇인지 한눈에 파악 가능

2. **심화 통계 간격 조정**
   - 항목 간 행 높이: 24px → 28px
   - 카드 높이: 120px → 132px
   - 비좁았던 2열 레이아웃에 여유 확보
