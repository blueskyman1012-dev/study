# 전체 UI 점검 및 재배치

> 작성일: 2026-02-09

## 수정 화면별 상세

### 1. BattleScreen (전투 화면) — 수직 간격 재배치
**문제**: 선택지(y=563) → 타이머(y=575) 간격 12px로 빡빡, 하단 버튼도 밀착
**수정**:
- 몬스터 영역: y=100→90, h=180→168 (12px 절약)
- 문제 카드: y=300→272, h=140→128 (40px 절약)
- 선택지: y=455→415, h=50→48 (약간 콤팩트)
- 타이머: y=575→545 (선택지→타이머 간격 12px→22px)
- 하단바: y=610→600, 버튼 h=45→46, fontSize 13→14
- 전체적으로 50px+ 여유 확보

### 2. RegisterScreen (오답 등록 화면) — 취소 버튼 위치
**문제**: 과목 버튼(y≈485)과 취소 버튼(y=560) 사이 75px 공백 과다
**수정**: 취소 버튼 y=560→510 (25px 감소)

### 3. MainScreen (메인 화면) — 하단 메뉴 균형
**수정**:
- 4개 메뉴 버튼: 하드코딩 → 변수 기반(menuBtnW=85, menuBtnH=55, gap=8)
- 버튼 높이 60→55px로 축소 (AI 버튼과의 간격 확보)
- AI 버튼: y=560→555, h=50→48
- 슬로건 폰트: 15px→14px

### 4. ShopScreen 강화탭 — 링 내부 가독성
**수정**:
- 링 안 퍼센트 텍스트: 10px→11px bold
- 아이콘 위치: ringCY-12→ringCY-14 (퍼센트 텍스트와 간격 확보)
- 퍼센트 위치: ringCY+8→ringCY+6 (링 영역 내 중심 정렬)

### 5. StatsScreen 종합탭 — 3열 카드 여유 확보
**수정**:
- 3열 카드 높이: 76→82px (+6px 여유)
- 내부 콘텐츠 Y 위치 전체 +2px 하향 (카드 내 상하 균형)
- 라벨 폰트: 8px→9px (가독성 향상)
