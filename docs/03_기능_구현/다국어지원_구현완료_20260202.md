# 다국어 지원 (i18n) 구현 완료 보고서

## 개요
설정 화면에서 언어를 변경할 수 있는 i18n 시스템 구현 완료.
지원 언어: 한국어(기본), English, 日本語, 中文

## 구현 내용

### 새 파일
- `app/src/i18n/i18n.js`: 번역 사전 + `t()`, `setLanguage()`, `getLanguage()`, `applyToHTML()` 함수
  - 약 190개 키에 대해 4개 언어 번역
  - `localStorage`의 `app_language` 키로 언어 설정 저장
  - `t(key, ...args)` 함수로 `{0}`, `{1}` 등 플레이스홀더 치환 지원
  - `applyToHTML()` 함수로 index.html의 DOM 텍스트도 동적으로 번역
  - `setLanguage()` 호출 시 `applyToHTML()` 자동 실행

### 수정된 파일

1. **화면 파일 (8개)**
   - `MainScreen.js` - 타이틀, 버튼, 슬로건, AI 상태
   - `BattleScreen.js` - 스테이지, 정답률, 타이머, 버튼, 아이템 드랍
   - `ResultScreen.js` - 클리어/실패, 통계 라벨
   - `ShopScreen.js` - 상점 UI 텍스트, 업그레이드/아이템 이름/설명
   - `StatsScreen.js` - 통계 라벨, 난이도명, 과목명
   - `SettingsScreen.js` - 설정 텍스트 + 언어 선택 UI 추가
   - `DungeonSelectScreen.js` - 던전 선택 텍스트
   - `RegisterScreen.js` - 등록 화면 텍스트, 과목명

2. **매니저 파일 (4개)**
   - `Game.js` - alert/prompt/confirm 메시지, AI 분석 메시지, 난이도 파싱, 내보내기/불러오기
   - `BattleManager.js` - 전투 메시지 (힌트, 부활, 시간연장 등)
   - `ShopManager.js` - 구매 메시지
   - `PlayerManager.js` - 레벨업 메시지

3. **기타 파일**
   - `constants.js` - `name`→`nameKey`, `description`→`descKey` 키 기반으로 변경
   - `MonsterManager.js` - 몬스터 이름 24종 번역, topic 제거(번역 불가 자유텍스트), API 폴백값 번역
   - `ItemManager.js` - 아이템 이름 15종 번역
   - `main.js` - 로그인/오류 메시지 번역, Google 로드 실패 메시지, 시작 시 `applyToHTML()` 호출

### 번역 범위
- **UI 텍스트**: 모든 화면의 라벨, 버튼, 제목
- **게임 콘텐츠**: 몬스터 이름 24종, 아이템 이름 15종, 보스명
- **시스템 메시지**: alert, confirm, prompt 다이얼로그
- **HTML DOM**: 로그인 화면 제목/슬로건, 오답등록/로그아웃/촬영/닫기 버튼
- **API 폴백값**: `'수학'`→`t('defaultTopic')`, `'문제'`→`t('defaultProblem')`, `'객관식'`→`t('multipleChoice')` 등
- **에러 메시지**: Google 로그인 실패, AI 분석 실패 등

### 번역 제외 (불가 항목)
- `monster.question` / `monster.answer`: 사용자가 촬영/등록한 원본 문제 텍스트 (자유입력)
- `monster.topic`: AI가 분석한 주제명 (DB에 원본 언어로 저장됨)
- `console.log` / `console.error`: 개발자 디버그 메시지 (사용자에게 노출 안됨)
- 코드 주석: 사용자에게 노출 안됨

### 언어 선택 UI (설정 화면)
- 사운드 설정 아래에 "🌐 언어 설정" 섹션 추가
- 4개 언어 버튼: 한국어 / English / 日本語 / 中文
- 현재 선택된 언어는 보라색 하이라이트
- 선택 시 즉시 적용 (canvas + HTML DOM 모두)
- 새로고침 후에도 localStorage에서 복원
