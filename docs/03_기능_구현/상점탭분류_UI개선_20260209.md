# 상점 탭 분류 + UI/UX 개선

## 작업일: 2026-02-09

## 개요
상점 화면에 3개 탭(강화/아이템/꾸미기)을 추가하고 카드 UI를 개선하여 가독성과 사용성을 향상시킴.

## 수정 파일

### 1. app/src/game/screens/ShopScreen.js
- 전면 리팩터링: 탭 시스템 도입 (SettingsScreen.js 패턴 동일 적용)
- 3개 탭: ⚡ 강화 / 🎒 아이템 / 🎨 꾸미기
- `renderShopFixedHeader()` 함수 추가 export (Game.js 고정 헤더용)
- UI 개선사항:
  - 카드 높이: 80→90px
  - 왼쪽 원형 아이콘 영역 (36x36, 22px 폰트)
  - 구매 가능 카드에 왼쪽 3px 악센트 테두리
  - 구매 불가 시 카드에 어두운 오버레이 (rgba(0,0,0,0.35))
  - 강화 프로그레스 바: 90→130px
  - MAX 달성 시 체크마크(✓ MAX) + 초록 테두리

### 2. app/src/i18n/i18n.js
- 4개 언어에 탭 번역 키 추가:
  - `shopTabUpgrade`: 강화 / Upgrades / 強化 / 强化
  - `shopTabConsume`: 아이템 / Items / アイテム / 道具
  - `shopTabCosmetic`: 꾸미기 / Cosmetic / デコ / 装饰

### 3. app/src/game/Game.js
- `renderShopFixedHeader` import 추가
- SHOP 화면 render에서 고정 헤더를 86px(헤더 50 + 탭바 36)로 확장
- `_getFixedHeaderHeight()` 메서드 추가 (화면별 고정 헤더 높이 반환)
- 스크롤 오프셋 임계값을 화면별로 동적 적용 (상점: 86px, 기타: 60px)

## 탭별 콘텐츠
- **⚡ 강화**: hp, time, goldBonus, damage (UPGRADES 4개)
- **🎒 아이템**: reviveTicket, hintTicket, timeBoost, doubleGold (4개)
- **🎨 꾸미기**: randomBg (1개)

## 검증
- `npm run build` 성공 확인
