# 통계 & 설정 화면 UI 개선

## 작업 일자
2026-02-09

## 변경 개요
통계 화면에 3탭 시스템을 도입하고, 설정 화면의 고정 헤더를 Game.js와 통합하여 UI 겹침 문제를 해결함.

## 변경 사항

### 1. StatsScreen.js — 3탭 시스템 도입
- **탭0 (📊 종합)**: 히어로 배너(승률 링), 연승&시간, 플레이 기록(승/패/콤보/골드), 아이템 사용 통계
- **탭1 (📚 과목)**: 과목별 정답률 링+바 차트, 과목별 오답 분석(정답/오답 바), 유형(topic)별 오답 TOP8
- **탭2 (🎯 분석)**: 난이도별 정답률+문제수 그래디언트 바, 난이도별 오답 바, 심화 통계, 최근 전적 그래프+런 목록
- `renderStatsFixedHeader(game)` 함수 export → Game.js에서 고정 헤더로 사용
- 고정 헤더 높이: 96px (제목 + 뒤로가기 + 탭 바)
- 콘텐츠 시작 Y: 106px

### 2. SettingsScreen.js — 고정 헤더 분리
- `renderSettingsFixedHeader(game)` 함수 export → Game.js에서 고정 헤더로 사용
- 기존 본문 내 헤더/탭바 렌더링 제거, 고정 헤더 함수로 이동
- 고정 헤더 높이: 96px (제목 + 뒤로가기 + 탭 바)
- API 탭 카드 높이 최적화 (120→110, 155→145, 90→80 등 여백 축소)

### 3. Game.js — 고정 헤더 연결
- `renderStatsFixedHeader` import 추가
- `renderSettingsFixedHeader` import 추가
- STATS 화면: 인라인 헤더 → `renderStatsFixedHeader` 호출로 변경
- SETTINGS 화면: 인라인 헤더 → `renderSettingsFixedHeader` 호출로 변경

### 4. i18n.js — 탭 번역 키 추가
- `stats_tabOverview`: 종합 / Overview / 総合 / 综合
- `stats_tabSubject`: 과목 / Subject / 科目 / 科目
- `stats_tabAnalysis`: 분석 / Analysis / 分析 / 分析
