# 런 상세 통계 확장 구현

## 개요
한 런(run)에서의 상세 행동 데이터를 추적하고, 결과 화면과 전체 통계 화면에서 시각적으로 표시하는 기능을 추가.

## 수정 파일 (6개)

### 1. `app/src/game/Game.js`
- `currentRun` 객체에 새 필드 추가:
  - `hintCount`: 힌트 사용 횟수
  - `timeBoostCount`: 시간 연장 사용 횟수
  - `reviveCount`: 부활권 사용 횟수
  - `timeoutCount`: 시간 초과 횟수
  - `wrongBySubject`: 과목별 오답 수 (예: `{ math: 3, english: 1 }`)
  - `correctBySubject`: 과목별 정답 수

### 2. `app/src/game/BattleManager.js`
- `onCorrectAnswer()`: 과목별 정답 카운트 증가
- `onWrongAnswer()`: 과목별 오답 카운트 증가
- `onTimeOut()`: `timeoutCount++`
- `useHint()`: 힌트 실제 사용 시 `hintCount++` (티켓/골드 모두)
- `useTimeBoost()`: `timeBoostCount++`
- 부활권 사용 (3곳): `reviveCount++`

### 3. `app/src/game/screens/ResultScreen.js`
- 카드 높이 확대 (230 -> 380)
- 구분선 + "아이템 사용" 섹션 추가
  - 스킵/힌트/시간+ (첫 줄)
  - 부활/시간초과 (둘째 줄)
- 과목별 정답/오답 미니 표시 (해당 런에서 풀었던 과목만)
- 버튼 위치 동적 조정

### 4. `app/src/game/StatsManager.js`
- `aggregateStats()` 루프에서 새 필드 합산:
  - `totalSkips`, `totalHints`, `totalTimeBoosts`, `totalRevives`, `totalTimeouts`
  - `avgSkipsPerRun`, `avgHintsPerRun`
  - `wrongBySubject` (전체 누적), `correctBySubject` (전체 누적)

### 5. `app/src/game/screens/StatsScreen.js`
- **H. 아이템 사용 통계**: 스킵/힌트/시간연장/부활/시간초과 미니카드 5개
- **I. 과목별 오답 분석**: 과목별 정답/오답 바 + 오답률 퍼센트
- `totalContentHeight`와 `scrollMaxY` 동적 업데이트

### 6. `app/src/i18n/i18n.js`
- 4개 언어(한/영/일/중) 번역 키 18개씩 추가:
  - `stats_itemUsage`, `stats_skipTotal`, `stats_hintTotal`, `stats_timeBoostTotal`
  - `stats_reviveTotal`, `stats_timeoutTotal`, `stats_perRun`
  - `stats_wrongBySubject`, `stats_wrongCount`, `stats_correctCount`, `stats_wrongRate`
  - `stats_hintShort`, `stats_timeBoostShort`, `stats_reviveShort`
  - `result_itemUsage`, `result_timeout`

## 검증
