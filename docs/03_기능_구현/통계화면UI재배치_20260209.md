# 통계 화면 UI 재배치

> 작성일: 2026-02-09

---

## 변경 내용

### 종합 탭 (탭0) 재배치

#### 기존
- 큰 히어로 배너(180px)에 승률 링 1개만 표시
- 아래에 3열 텍스트 요약
- 연승/시간 2열 카드
- 플레이 기록 2x2 그리드
- 아이템 사용 별도 카드

#### 변경
- **트리플 링 히어로(130px)**: 정답률(좌), 승률(중앙·크게), 최고콤보(우) 3개 원형 게이지 나란히 배치
- **3열 카드(76px)**: 승/패, 연승, 시간을 동일 크기 카드 3개로 한 줄에 배치
- **골드+아이템 통합 카드(110px)**: 골드 총 수입을 상단에 크게 표시, 구분선 아래에 아이템 사용 통계 배치
- 기존 플레이 기록 2x2 그리드 제거 (승/패 정보가 3열 카드로 통합)
- 전체 높이 약 80px 절약

### 분석 탭 (탭2) 재배치

#### 기존
- 난이도별 정답률 카드
- 난이도별 오답 카드 (별도)
- 심화 통계
- 최근 전적 그래프 + 런 목록 (최하단)

#### 변경
- **최근 전적 그래프를 최상단**으로 이동 (가장 관심 있는 데이터 먼저)
- **런 목록을 그래프 바로 아래** 별도 카드로 분리 (그래프와 목록 간 간섭 제거)
- **난이도별 통합**: 정답률 + 오답 수를 하나의 카드에 합침 (바 옆에 오답 수 표시)
- 심화 통계 행 간격 28→24px로 압축
