# 레벨 100 시스템 구현

## 날짜
2026-02-02

## 요구사항
- UI에는 최대 레벨 99로 표시
- 실제로는 레벨 100까지 도달 가능
- 레벨 100 달성 시 특별 보너스: 공격력+10, HP+10, 시간+15

## 수정 내용

### constants.js
- `maxLevel`: 99 → 100 (실제 최대)
- `displayMaxLevel`: 99 추가 (UI 표시용)

### PlayerManager.js
- `getLevelBonusHp()`: 레벨 100 시 추가 HP +10
- `getLevelBonusDamage()`: 레벨 100 시 추가 공격력 +10
- `getLevelBonusTime()`: 레벨 100 시 추가 시간 +15
- `gainExp()`: 레벨 100 도달 시 특별 보너스 메시지 표시

### MainScreen.js
- 레벨 100일 때 레벨 숫자 대신 금색 "MAX" 표시
- EXP바 꽉 차고 금색, "MAX" 텍스트 표시

### BattleScreen.js
- 레벨 100일 때 "LV.100" 대신 금색 "MAX" 표시

### Game.js (showLevelProgress)
- UI에 최대 레벨 99로 표시
- 레벨 100 보너스 정보를 다음 보너스 영역에 안내
- 레벨 100 달성 후에는 MAX 보너스 내용 표시

### i18n (4개 언어)
- `maxLevelBonus` 번역 키 추가 (한/영/일/중)

## 수정 파일
