# 몬스터 이름이 문제와 연관있게 생성되도록 수정

## 날짜
2026-02-02

## 문제
몬스터 이름이 `stage % 6`으로 순환 배정되어, 문제의 주제(topic)나 내용과 전혀 무관한 이름이 표시되었음.
예: 일차방정식 문제인데 "거미", 화학반응 문제인데 "슬라임" 등.

## 수정 내용

### MonsterManager.js - `_assignMonsterAppearance()` 수정
- **topic 기반 이름 생성**: monster에 topic이 있으면 `"{topic} {몬스터이름}"` 형태로 표시
  - 예: "일차방정식 슬라임", "화학반응 해골병사"
- **키워드 fallback**: topic이 없으면 monster의 keywords 배열에서 첫 번째 키워드를 접두어로 사용
- **문제 텍스트 추출**: keywords도 없으면 문제 텍스트에서 괄호/따옴표 안의 핵심어 추출
- **난이도 기반 아이콘**: 난이도에 따라 다른 몬스터 외형을 보여주도록 offset 적용
  - 쉬움(1): 슬라임/해골 계열
  - 보통(2): 박쥐/좀비 계열
  - 어려움(3): 유령/거미 계열

### 보스 몬스터도 topic 반영
- topic이 있으면 `"👹 일차방정식 보스"` 형태로 표시
- topic이 없으면 기존과 동일하게 `"👹 보스"` 표시

### 새로 추가된 메서드
- `_extractNamePrefix(monster)`: 문제에서 이름 접두어를 추출하는 헬퍼

## 수정 파일
- `app/src/game/MonsterManager.js`
