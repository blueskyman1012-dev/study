# UI 투명도 설정 구현 완료

## 변경일: 2026-02-02

## 기능 설명
설정 화면에서 UI 요소의 투명도를 30%~100% 범위로 조절할 수 있는 설정 추가.

## 조작 방법
- 슬라이더 바 클릭: 원하는 위치 클릭으로 투명도 직접 설정
- -/+ 버튼: 10% 단위로 투명도 감소/증가
- 범위: 30% ~ 100% (최소 30%로 제한하여 UI 가독성 보장)

## 적용 범위
- 메인 화면, 설정, 상점, 업적, 통계, 던전 선택, 등록 화면에 적용
- 전투/결과 화면은 가독성을 위해 항상 100% 유지

## 수정 파일
- `app/src/canvas/Renderer.js` - uiOpacity 관리 (localStorage 저장/로드)
- `app/src/game/Game.js` - 렌더링 시 UI 투명도 적용/해제
- `app/src/game/screens/SettingsScreen.js` - 투명도 슬라이더 UI 추가
- `app/src/i18n/i18n.js` - 4개 언어 번역 키 추가

## 추가 변경
- 업적 추가: 공격력 100 달성(+ATK1), 체력 500 달성(+HP1)
- handleInput에서 callback에 클릭 좌표 전달하도록 개선
