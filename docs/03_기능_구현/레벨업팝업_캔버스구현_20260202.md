# 레벨업 캔버스 팝업 구현 완료

## 작업일: 2026-02-02

## 변경 내용

### 문제
- 레벨업 시 브라우저 `alert()` 사용으로 게임 루프 블로킹
- 모바일 UX 저하

### 해결
캔버스 위에 오버레이 팝업으로 레벨업 표시. 2.5초 표시 후 0.5초 페이드아웃. 탭하면 즉시 닫힘.

### 수정 파일

#### 1. `app/src/game/EffectSystem.js`
- `levelUpPopup` 상태 추가
- `showLevelUp(level, bonusMsg)` 메서드: 팝업 표시 + 금색 파티클 폭발 + 플래시
- `update()`에서 타이머 관리 (2.5초 표시 → 0.5초 페이드아웃)
- `render()`에서 팝업 렌더링:
  - 반투명 어두운 오버레이
  - 그라데이션 배경 + 둥근 모서리 카드
  - "LEVEL UP!" 금색 텍스트 + 검은 윤곽선
  - "LV.{n}" 보라색 텍스트
  - 보너스 메시지 (HP+, 공격력+, 시간+)
- `isLevelUpActive()` / `dismissLevelUp()` 메서드

#### 2. `app/src/game/PlayerManager.js`
- `constructor`에 `game` 참조 추가
- `gainExp()`에서 `alert()` 제거 → `this.game.effects.showLevelUp()` 호출

#### 3. `app/src/game/Game.js`
- `PlayerManager` 생성 시 `this` 전달
- `handleInput()`에서 팝업 활성 시 탭하면 닫기 처리
