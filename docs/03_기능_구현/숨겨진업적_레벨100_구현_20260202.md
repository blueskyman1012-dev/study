# 숨겨진 업적: 레벨 100 비밀 업적 구현

## 날짜
2026-02-02

## 요구사항
- 레벨 99 업적 바로 밑에 숨겨진 업적 배치
- 달성 전에는 "???" / "숨겨진 업적"으로 표시
- 레벨 100 달성 시 해제되며 골드 10,000 보상
- 해제 후에는 금색 테두리로 특별하게 표시

## 수정 내용

### constants.js
- `level_100` 업적 추가 (`hidden: true`, reward: 10000)
- level_99 바로 뒤에 배치

### AchievementManager.js
- `onLevelUp()`: 레벨 100 도달 시 `level_100` 업적 해제

### AchievementScreen.js
- 숨겨진 업적(`hidden: true`) 미해제 시:
  - 아이콘: ❓, 이름: ???, 설명: "숨겨진 업적", 보상: ????G
  - 어두운 배경으로 존재감만 암시
- 숨겨진 업적 해제 시:
  - 금색 배경 + 금색 테두리로 특별 표시
  - 이름 금색으로 렌더링
- 업적 총 개수: 숨겨진 업적은 해제 전까지 총 개수에서 제외

### i18n (4개 언어)
- `achv_level_100`: 전설의 시작 / Legend Begins / 伝説の始まり / 传说的开始
- `achv_level_100_desc`: 레벨 100 달성 — 한계를 넘어선 자
- `hiddenAchievement`: 숨겨진 업적 / Hidden Achievement / 隠された実績 / 隐藏成就
