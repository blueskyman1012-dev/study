# 던전 입장 시 문제 자동 채우기 (2026-02-09)

## 기능 설명
- 던전 입장 시 문제(몬스터)가 **30개 미만**이면 AI가 자동으로 부족분을 생성
- 난이도: **쉬움(1) 또는 보통(2)** 랜덤
- 주제: 해당 과목의 주제 중 랜덤 선택
- API 키가 없으면 기존처럼 보유 문제로만 진행

## 동작 흐름
1. `startDungeon(subject)` 호출
2. 과목별 몬스터 로드 → 개수 확인
3. 30개 미만 AND AI API 키 있음 → `_autoFillProblems()` 실행
4. 로딩 화면 표시 ("문제 자동 생성 중... N/M개 생성 완료")
5. 5개씩 배치 생성 (API 부하 분산)
6. 생성된 문제를 DB에 저장 → 몬스터 재로드
7. 정상적으로 던전 시작

## 생성 규칙
- SmilePrint API 우선, 없으면 Gemini 폴백
- 배치당 5개씩 생성 (API 호출 최소화)
- 난이도 1~2 랜덤 (50% 확률)
- 생성 실패 시 중단하고 보유 문제로 진행
- 문제 0개 + API 키 없으면 기존과 동일하게 "문제 없음" 알림
