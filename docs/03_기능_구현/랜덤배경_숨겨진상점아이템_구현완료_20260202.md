# 랜덤 배경 숨겨진 상점 아이템 구현 완료

## 날짜
2026-02-02

## 개요
상점 화면 하단에 방향키로만 접근 가능한 숨겨진 "랜덤 배경" 아이템 추가.

## 구현 내용

### 1. 배경 테마 시스템 (Renderer.js)
- 7개 배경 테마: default, emerald, crimson, amber, violet, cyan, rose
- `BG_THEMES` 배열로 각 테마의 bg/grid/gridSub/glow 색상 정의
- `setRandomBgTheme()`: 현재 테마 제외 랜덤 선택, localStorage 저장
- `_loadBgTheme()`: 앱 시작 시 저장된 테마 로드
- 그리드 캔버스 캐싱 자동 갱신

### 2. 상점 스크롤 지원 (Game.js, main.js)
- `scrollY`, `scrollMaxY` 속성 추가
- 상점 화면에서 `ctx.translate(0, -scrollY)` 적용
- 헤더(골드 표시)는 고정 렌더링
- 스크롤 인디케이터(↓ 화살표) 표시
- 방향키(ArrowUp/ArrowDown) 40px 단위 스크롤
- 화면 전환 시 scrollY 초기화

### 3. 랜덤 배경 아이템 (ShopScreen.js)
- 소비 아이템 섹션 아래 40px 간격 후 배치
- 보라색 테마 카드 (rgba(147,51,234,0.15) + #9333ea 보더)
- 가격: 300G
- 구매 시: 골드 차감 → 저장 → 배경 변경 → 보라색 플래시 → 모달 알림

### 4. 다국어 지원 (i18n.js)
- randomBg, randomBgDesc, randomBgApplied 키 4개 국어 추가 (한/영/일/중)

## 수정 파일
- `app/src/canvas/Renderer.js` — BG_THEMES, setRandomBgTheme, _loadBgTheme
- `app/src/game/Game.js` — 스크롤 속성, 상점 스크롤 렌더링
- `app/src/main.js` — 방향키 스크롤 이벤트
- `app/src/game/screens/ShopScreen.js` — 랜덤 배경 카드 + scrollMaxY 설정
- `app/src/i18n/i18n.js` — 4개 국어 번역 키 추가
