## 변경 내용

### Tab 1 (과목) 섹션 B 신설: 유형별 오답률 TOP

기존 구조: A(과목별 정답률) → B(과목별 유형 상세)
변경 구조: A(과목별 정답률) → **B(유형별 오답률 TOP)** → C(과목별 유형 상세)

#### 기능 설명
- 전 과목의 유형 데이터를 통합하여 **오답률 기준 내림차순** 정렬
- 최대 5개까지 취약 유형을 한눈에 표시
- 2회 이상 시도 & 오답 1회 이상인 유형만 표시 (노이즈 필터)

#### 각 항목 표시 내용
| 요소 | 설명 |
|------|------|
| 과목 아이콘 + 유형명 | 어떤 과목의 어떤 유형인지 표시 |
| 오답률 바 | 120px 바, 60%↑ 빨강, 40%↑ 노랑 |
| 오답률 % | 숫자로 표시 |
| ❌오답수/시도수 | 구체적인 오답 통계 |

#### 데이터 소스
- `stats.subjectTopics` (StatsManager에서 집계)
- 이미 수집되고 있는 `attempts`/`correct` 데이터 활용
- `wrong = attempts - correct`, `wrongRate = wrong / attempts * 100`
