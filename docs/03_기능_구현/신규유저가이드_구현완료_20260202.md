# 신규 유저 가이드 구현 완료

## 개요
첫 접속 유저에게 캔버스 오버레이 형태의 4페이지 가이드를 표시하는 기능 구현.

## 수정 파일

### 1. `app/src/game/Game.js`
- `guideStep` 상태 추가 (null이면 비활성, 0~3이면 해당 페이지)
- `init()` 끝에서 `totalRuns === 0 && !localStorage.getItem('guide_shown')` 체크
- `handleInput()`에서 가이드 활성 시 탭하면 다음 페이지 또는 닫기
- `render()`에서 가이드 오버레이 렌더링
  - 반투명 어두운 배경 + 중앙 카드(340x520)
  - 페이지별 아이콘/제목/설명
  - 페이지 인디케이터 (● ○ ○ ○)
  - 하단 버튼 ("다음 ▶" / "시작하기!")

### 2. `app/src/i18n/i18n.js`
- 4개 언어(ko/en/ja/zh)에 가이드 텍스트 키 추가
  - `guide_title_0` ~ `guide_title_3`: 각 페이지 제목
  - `guide_desc_0` ~ `guide_desc_3`: 각 페이지 설명
  - `guide_next`: 다음 버튼 텍스트
  - `guide_start`: 시작 버튼 텍스트

## 가이드 페이지 구성 (4페이지)
1. 환영 (👋) — 오답헌터 게임 소개
2. 오답 등록 (📸) — 문제 사진 촬영 → 몬스터 생성
3. 던전 전투 (⚔️) — 던전 입장 → 문제 풀기 → 몬스터 처치
4. 성장 (📈) — 경험치/골드 → 레벨업/상점 강화

## 신규 유저 판별 조건
- `!localStorage.getItem('guide_shown')` (가이드를 아직 안 본 유저)
- 서버 동기화로 totalRuns가 복원될 수 있어 localStorage만으로 판별

## 검증
- `npm run build` 성공 확인
- 가이드 닫으면 `localStorage`에 `guide_shown=1` 저장되어 재표시 안 됨
