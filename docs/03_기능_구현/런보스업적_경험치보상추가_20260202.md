# 런/보스 업적 경험치 보상 추가

## 날짜
2026-02-02

## 변경 내용

### 수정 파일
- `app/src/utils/constants.js` - 런/보스 업적에 `rewardExp` 필드 추가
- `app/src/game/AchievementManager.js` - `unlock()` 메서드에서 경험치 지급 처리
- `app/src/game/screens/AchievementScreen.js` - 보상 표시에 XP 포함

### 경험치 보상 값

| 업적 | 골드 | 경험치 |
|---|---|---|
| first_clear (첫 클리어) | 200G | 30 XP |
| clear_10 (20회 클리어) | 500G | 80 XP |
| clear_50 (80회 클리어) | 2000G | 200 XP |
| perfect_run (퍼펙트 런) | 1000G | 150 XP |
| boss_normal (일반 보스) | 100G | 50 XP |
| boss_mid (중간 보스) | 300G | 120 XP |
| boss_final (최종 보스) | 500G | 300 XP |

### 구현 방식
- 업적 데이터에 `rewardExp` 필드 추가 (선택적 - 없으면 경험치 미지급)
- `AchievementManager.unlock()`에서 `rewardExp`가 있으면 `playerManager.gainExp()` 호출
- 업적 화면에서 경험치 보상이 있는 업적은 `200G +30XP` 형태로 표시
