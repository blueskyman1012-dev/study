# 업적 진척도 표시 구현 완료

## 날짜
2026-02-02

## 변경 내용

### 수정 파일
- `app/src/game/screens/AchievementScreen.js`

### 구현 사항
1. **`getProgress()` 헬퍼 함수 추가**: 업적 타입별로 플레이어 데이터에서 현재 진행도를 계산
2. **미해제 업적에 진행바 표시**: 너비 220px, 높이 8px 진행바 + `현재/목표` 텍스트
3. **카드 높이 조정**: 진행바가 있는 미해제 업적은 카드 높이 50→65로 증가

### 진행도 데이터 매핑
| 업적 type | player 데이터 소스 |
|---|---|
| combo | `player.stats.bestCombo` |
| clear | `player.stats.totalClears` |
| level | `player.level` |
| gold | `player.totalGoldEarned` |
| kills | `player.stats.totalKills` |
| subject | `player.subjectCounts.[과목]` |
| answers | `player.totalCorrectAnswers` |
| boss | 진행바 없음 (달성 여부만) |
| perfect_run | 진행바 없음 (달성 여부만) |

### 동작 규칙
- 해제된 업적: 기존대로 표시 (진행바 없음)
- boss/perfect_run 업적: 진행바 없이 표시
- 기타 미해제 업적: 진행바 + 숫자 표시
