# 과목별 문제유형 통계 기능

> 작성일: 2026-02-09

---

## 개요
통계 화면의 과목 탭에서 각 과목별 문제유형(topic)의 정답률, 오답률, 푼 횟수를 확인할 수 있도록 기능 추가.

## 변경 내용

### 1. StatsManager.js — 과목별 유형 데이터 집계
**변경**: 몬스터 루프에서 `subject`와 `topic`을 교차 참조하여 `subjectTopics` 필드 생성

**데이터 구조**:
```javascript
stats.subjectTopics = {
  math: {
    '인수분해': { attempts: 5, correct: 3 },
    '방정식': { attempts: 8, correct: 6 },
    ...
  },
  english: {
    'grammar': { attempts: 4, correct: 2 },
    ...
  }
}
```

### 2. StatsScreen.js — 과목 탭 UI 변경
**변경**: 기존 "유형별 오답 TOP8" (글로벌) → 과목별 문제유형 상세 카드

**UI 구성 (과목별 카드)**:
- 과목 아이콘 + 이름 헤더
- 각 유형별 행:
  - 유형명 (8자 초과 시 말줄임)
  - 정답/오답 비율 미니 바 (초록/빨강)
  - 푼 횟수
  - 정답률 (%)
  - 오답률 (%)
- 푼 횟수 기준 내림차순 정렬

### 3. i18n.js — 번역 키 추가
| 키 | 한국어 | 영어 | 일본어 | 중국어(간체) |
|---|---|---|---|---|
| stats_topicDetail | 유형별 | By Type | 類型別 | 按类型 |
| stats_colAttempts | 횟수 | Count | 回数 | 次数 |
| stats_colCorrectRate | 정답률 | Correct% | 正答率 | 正确率 |
| stats_colWrongRate | 오답률 | Wrong% | 誤答率 | 错误率 |
