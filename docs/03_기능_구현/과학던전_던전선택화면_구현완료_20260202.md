# 과학 던전 + 던전 선택 화면 구현 완료

## 개요
기존 '던전 입장' 클릭 → 바로 전투 흐름을, '던전 입장' → **던전 선택 화면** → 수학/과학 중 택1 → 전투로 변경.
과학 문제 AI 생성 기능 추가 (중+고 혼합). 능력치는 공유, 문제와 몬스터 이름만 다름.

## 변경 파일

### 1. `app/src/game/screens/DungeonSelectScreen.js` (신규)
- 던전 선택 화면 렌더링
- 수학 던전 카드 (📐 수학 던전) + 과학 던전 카드 (🔬 과학 던전)
- 각 카드에 해당 과목 몬스터 수 표시
- 뒤로가기 버튼 → MAIN
- 클릭 시 `game.startDungeon('math')` 또는 `game.startDungeon('science')`

### 2. `app/src/services/ProblemGeneratorService.js` (수정)
- `SCIENCE_TOPICS` 상수 추가 (8개 주제: 중학 4 + 고등 4)
  - 중: 힘과 운동, 화학반응, 세포, 지구과학
  - 고: 역학, 전자기, 몰과 반응식, 유전
- `buildSciencePrompt()` 메서드 추가 — "한국 과학 선생님" 프롬프트
- `generateProblems()` — subject 파라미터 추가, 과학이면 과학 프롬프트 사용
- `generateRandom()` — subject에 따라 토픽 소스 변경
- `getTopics()` — subject에 따라 MATH_TOPICS 또는 SCIENCE_TOPICS 반환

### 3. `app/src/game/Game.js` (수정)
- `renderDungeonSelectScreen` import 추가
- `render()` switch에 `case SCREENS.DUNGEON_SELECT` 추가
- `startDungeon(subject)` 수정: subject별 몬스터 로드, AI 생성 시 subject 전달
- `_doAIGenerateMenu()`: 수학/과학 과목 선택 단계 추가

### 4. `app/src/game/MonsterManager.js` (수정)
- `loadMonstersBySubject(subject)` 메서드 추가
  - math: subject 없는 몬스터도 수학으로 간주
  - science: subject === 'science'인 것만 필터

### 5. `app/src/game/screens/MainScreen.js` (수정)
- '던전 입장' 클릭 → `game.changeScreen(SCREENS.DUNGEON_SELECT)`로 변경
- AI 생성 버튼 텍스트: '🤖 AI 문제 생성'으로 변경
