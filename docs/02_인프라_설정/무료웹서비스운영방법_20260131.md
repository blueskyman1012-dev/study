# PC 기반 웹 서비스 24시간 무료 운영 방법

## 1. Oracle Cloud Free Tier (팩트 체크 완료)

### 스펙 (영구 무료 - Always Free)
- **ARM(Ampere A1)**: 4 OCPU, 24GB RAM, 200GB 블록 스토리지
- **AMD VM**: 2대 (각 1/8 OCPU, 1GB RAM)
- **네트워크**: 월 10TB 아웃바운드 전송, 고정 IP 1개
- **기간**: 영구 무료 (AWS 12개월 제한과 다름)

### 주의사항
- 인기 리전(미국, 프랑크푸르트 등)은 ARM 인스턴스 품절이 잦음
- Home Region은 가입 후 변경 불가
- 방화벽/네트워크/구획(Compartment) 설정이 복잡함
- 리눅스 서버 관리 지식(SSH, 명령어)이 필요함
- 인스턴스 삭제 우려가 있음 (드물지만 보고됨)

### 난이도: ★★★★☆ (IT 초보에게 어려움)

## 2. Cloudflare Pages vs Vercel vs Netlify 비교

### 무료 요금제 비교표

| 항목 | Cloudflare Pages | Vercel (Hobby) | Netlify (Free) |
|------|-----------------|----------------|----------------|
| **대역폭** | **무제한** | 100GB/월 | 100GB/월 |
| **빌드** | 500회/월 | 6,000분/월 | 300분/월 |
| **서버리스 함수** | 10만 요청/일 | 100만 엣지/월 | 12.5만/월 |
| **HTTPS** | 자동 | 자동 | 자동 |
| **CDN** | 전세계 최대급 | 100+ 위치 | 다수 위치 |
| **커스텀 도메인** | 100개/프로젝트 | 지원 | 지원 |

### 각 플랫폼 장단점

#### Vercel ⭐ 예비 고1 추천 1순위
- **장점**
  - GitHub 연동 후 클릭 3번이면 배포 완료
  - Next.js/React 프로젝트 자동 감지, 설정 불필요
  - Push하면 자동 재배포 (CI/CD)
  - 한글 가이드/블로그가 매우 많음
  - 프리뷰 배포로 변경사항 미리 확인 가능
- **단점**
  - 무료는 개인 프로젝트만 가능 (상업용 X)
  - 대역폭 100GB 제한
  - 서버(백엔드) 기능은 제한적

#### Netlify
- **장점**
  - 드래그앤드롭으로 폴더 끌어다 놓으면 배포됨 (가장 쉬움)
  - 폼 처리, 인증 기능 내장
  - 직관적인 관리 화면
- **단점**
  - 빌드 300분 제한이 좀 적음
  - 2025년 크레딧 시스템 도입으로 무료 범위 축소 추세

#### Cloudflare Pages
- **장점**
  - **무제한 대역폭** (트래픽 걱정 없음)
  - 세계 최대급 CDN으로 속도 빠름
  - Workers와 결합하면 풀스택 가능
- **단점**
  - 설정이 Vercel/Netlify보다 약간 복잡
  - 한글 자료가 상대적으로 적음
  - 초보자 친화적이지 않은 대시보드

## 3. 예비 고1 입장 최종 추천

### 🏆 1순위: Vercel
- **이유**: GitHub에 코드 올리고 → Vercel에서 Import → Deploy 끝
- 서버 관리 지식 전혀 불필요
- 한글 블로그 튜토리얼이 넘침
- React, Next.js 배우면서 자연스럽게 사용 가능

### 2순위: Netlify
- **이유**: 폴더를 드래그앤드롭하면 바로 배포
- GitHub 없어도 사용 가능
- Vercel보다 더 단순한 시작 가능

### 3순위: Cloudflare Pages
- **이유**: 트래픽이 많아질 것 같으면 무제한 대역폭이 매력적
- 다만 초기 설정이 위 둘보다 복잡

### ❌ Oracle Cloud는 비추천 (예비 고1 기준)
- 리눅스 서버 관리, SSH, 방화벽 설정 등 IT 지식 많이 필요
- 설정 실수 시 보안 문제 발생 가능
- 문제 해결 시 영어 문서 의존도 높음
- **단, 백엔드 서버(DB, API)가 반드시 필요하면** 나중에 도전할 가치 있음

## 4. 서비스 유형별 추천

| 만들고 싶은 것 | 추천 플랫폼 | 비고 |
|---------------|-----------|------|
| 포트폴리오/블로그 | Vercel 또는 Netlify | 가장 쉬움 |
| React/Next.js 웹앱 | Vercel | 최적화됨 |
| 간단한 HTML/CSS 사이트 | Netlify (드래그앤드롭) | 코딩 입문자용 |
| API 서버 + DB 필요 | Vercel + Supabase(무료) | 풀스택 가능 |
| 트래픽 많은 서비스 | Cloudflare Pages | 대역폭 무제한 |
| 게임 서버/커스텀 서버 | Oracle Cloud | IT 지식 필요 |

## 5. PC 직접 운영 (참고)

PC에서 꼭 돌려야 한다면:
- **Cloudflare Tunnel**: 포트포워딩 없이 외부 접속 (무료)
- **NSSM** (Windows): 프로그램을 서비스로 등록, 자동 재시작
- **UptimeRobot**: 다운 시 알림 (무료)

단, 정전/인터넷 장애 시 중단되므로 클라우드 서비스 권장.
