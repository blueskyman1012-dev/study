# 크로스 기기 동기화 데이터 누락 수정 (2026-02-09)

## 문제 상황
- 서로 다른 기기에서 같은 계정으로 로그인하면 일부 데이터가 동기화되지 않음
- 일일/주간 미션 진행도, 업적, 코스메틱 설정 등이 기기마다 다름

## 원인
서버 업로드(`putPlayer`)와 다운로드(`downloadPlayerData`)에서 다음 필드들이 누락:

| 필드 | 설명 |
|---|---|
| `achievements` | 해금된 업적 목록 |
| `dailyMissions` | 일일 미션 날짜, 진행도, 보상수령 |
| `weeklyMissions` | 주간 미션 주차, 진행도, 보상수령 |
| `cosmetics` | 구매한 테마, 파티클, 데미지 텍스트 스타일 |
| `totalGoldEarned` | 누적 골드 획득량 |
| `totalCorrectAnswers` | 누적 정답 수 |
| `subjectCounts` | 과목별 등록 수 |
| `achievementBonusDamage` | 업적 보너스 공격력 |
| `achievementBonusHp` | 업적 보너스 HP |

## 수정 내용

### ApiService.putPlayer() - 서버 업로드
- `data` 객체에 9개 필드 추가

### ApiService.downloadPlayerData() - 서버 다운로드
- `extra`에서 9개 필드 복원 (존재하는 경우에만 덮어쓰기, 없으면 로컬 값 유지)
