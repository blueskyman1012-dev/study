# 업적 달성 조건과 이름 불일치 수정

## 날짜
2026-02-02

## 문제
업적 이름에 표시된 숫자와 실제 달성 조건(설명/진행바)이 다르게 표시되는 오류.
밸런스 조정으로 condition 값이 변경되었으나 이름이 업데이트되지 않았음.

## 불일치 목록 및 수정 내용

| 업적 ID | 수정 전 이름 | 실제 조건 | 수정 후 이름 |
|---|---|---|---|
| kills_50 | 50마리 처치 | 80마리 처치 | **80마리 처치** |
| kills_100 | 100마리 처치 | 200마리 처치 | **200마리 처치** |
| kills_500 | 학살자 | 800마리 처치 | **800마리 학살자** |

※ 콤보/레벨/골드/정답 업적은 이름이 별칭(예: "불꽃 시작", "금화 수집가")이라 혼동 없음.
  설명(desc)은 모두 실제 조건과 일치하여 수정 불필요.

## 수정 파일
- `app/src/i18n/i18n.js` (한국어/영어/일본어/중국어 4개 언어)
